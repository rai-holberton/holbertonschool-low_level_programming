# Hello, World!

Ce projet marque mes premiers pas dans l'univers du langage C. J'y explore le cycle de vie d'un programme, du code source jusqu'à l'exécutable final.

Objectifs d'apprentissage :
Comprendre le rôle du préprocesseur, du compilateur, de l'assembleur et de l'éditeur de liens.
Apprendre à utiliser les fonctions d'affichage standard (`printf`, `puts`, `putchar`).
Respecter la norme de codage Betty pour produire un code propre et lisible.
